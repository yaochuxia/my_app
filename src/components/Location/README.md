## 参数

### Location

| 参数 | 说明 | 类型 | 默认值| 备注 |
|------|------|-----|------|------|
| iconType | 图标类型 | String | material | - |
| iconName | 图标名称 | String | location-on | - |
| iconColor | 图标颜色 | String | #FFF | - |
| iconSize | 图标尺寸 | Number | 24 | - |
| onPress | 点击回调 | Function | () => null | - |
